# Software Engineering, BS

The Software Engineering, BS program provides a foundation in building reliable software systems. You will learn how to design, implement, test, and maintain software across modern platforms.

## Program overview

Coursework balances theory with hands-on practice. You will develop problem-solving skills, learn core engineering principles, and apply them to real-world software projects.

## What you will study

- Software design and development fundamentals
- Data structures and algorithms
- Testing and quality practices
- Collaboration and project workflows

## Learning outcomes

- Build and maintain software applications using modern practices
- Apply engineering principles to improve software quality
- Communicate effectively in technical teams

Online and campus options may be available. Ask an admissions advisor to confirm the best format for your goals.
