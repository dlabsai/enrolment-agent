# Cybersecurity, MS

The Cybersecurity, MS program builds advanced expertise in protecting systems, managing risk, and aligning security with organizational goals. You will develop deeper technical and strategic knowledge for complex security environments.

## Program overview

This program emphasizes advanced analysis, governance, and secure system design. You will explore how to evaluate security posture and guide improvements across people, processes, and technology.

## What you will study

- Advanced security architecture and design
- Risk management and governance practices
- Secure systems and application strategies
- Incident analysis and response planning

## Learning outcomes

- Evaluate organizational security posture and propose improvements
- Apply advanced security principles to complex systems
- Communicate security strategy to technical and non-technical stakeholders
