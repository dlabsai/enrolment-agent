# Cybersecurity, BS

The Cybersecurity, BS program focuses on protecting information systems and managing risk. You will study security principles, secure design, and practical techniques for safeguarding data.

## Program overview

Coursework emphasizes how to analyze systems, identify vulnerabilities, and design protective strategies. You will learn to think critically about risk, privacy, and responsible technology use.

## What you will study

- Network and systems security fundamentals
- Secure software and application practices
- Risk assessment and governance basics
- Incident response concepts

## Learning outcomes

- Identify common security risks and mitigation approaches
- Apply secure design principles to technology systems
- Communicate security considerations to technical and non-technical audiences
